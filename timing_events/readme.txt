# Attributions

This repository contains scripts that creates required input files to run "MutationTimeR" and "Moltime"
The scripts related to each sofware are in the separate directories with the name of software.
In the data folder there are some example data that can be used to run the scripts and create required inputs.

For more information about each package please refere to:
https://github.com/gerstung-lab/MutationTimeR
and
https://github.com/nicos-angelopoulos/mol_time


If you have any questions about the scripts please contact me at "shaghayegh.soudi@gmail.com"
